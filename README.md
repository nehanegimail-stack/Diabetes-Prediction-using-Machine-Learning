# Diabetes Prediction Using Machine Learning

## Project Overview
This project predicts whether a person is diabetic based on medical attributes such as glucose level, BMI, blood pressure, insulin, and age.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Models
- Logistic Regression
- Decision Tree
- Random Forest

## Results
The model achieved good accuracy and was evaluated using confusion matrix and classification report.
